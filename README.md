# Guia de Recomendações – Truque de Mestre (2013)

Descrição
Este projeto é um mini-site desenvolvido em HTML e CSS, apresentando informações e recomendações relacionadas ao filme Truque de Mestre (2013).  
A página exibe imagens, listas, tabela, links e uma breve explicação sobre o tema, funcionando como um guia básico de recomendações.

Tecnologias Utilizadas
- HTML
- CSS
